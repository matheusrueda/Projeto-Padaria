## 📚 Sobre o Projeto

O **Projeto Padaria Pão Dourado v2.0** é uma aplicação web moderna e responsiva que simula o site de uma padaria, desenvolvida para aplicar as melhores e mais atuais práticas de desenvolvimento front-end (2025). 

Esta versão representa uma evolução completa do projeto original, focando em performance, acessibilidade e experiência do usuário (UX).

## 🚀 Principais Funcionalidades e Tecnologias

- **Arquitetura Moderna:** JavaScript ES6+ orientado a objetos, manipulação avançada de DOM e `Intersection Observer API` para animações fluidas.
- **Design System e Estilização:** CSS moderno com Flexbox, Grid e mais de 80 Design Tokens (variáveis CSS) organizados, incluindo suporte automático aos modos Claro e Escuro.
- **Progressive Web App (PWA):** Aplicativo instalável com Service Worker, cache avançado e suporte offline.
- **Performance:** Lazy loading inteligente, recursos otimizados e foco total nas métricas de Web Vitals (LCP < 2.5s, FID < 100ms, CLS < 0.1).
- **Acessibilidade e SEO:** HTML Semântico, Microdados Schema.org, suporte completo a leitores de tela (ARIA Labels) e foco contínuo nas diretrizes WCAG 2.1.

## 📋 Como Executar

### 1. Servidor Local (Recomendado para PWA)
Clone o repositório e inicie um servidor HTTP na pasta raiz do projeto.
```bash
git clone https://github.com/matheusrueda/projeto-padaria.git
cd projeto-padaria

# Exemplo usando Python
python -m http.server 8000

# Exemplo usando Node.js
npx http-server .
```
Também é possível utilizar a extensão **Live Server** no VS Code.

### 2. Execução Simples
Apenas abra o arquivo `index.html` em seu navegador. (Nota: Recursos de PWA podem não funcionar corretamente sem um servidor HTTP).
